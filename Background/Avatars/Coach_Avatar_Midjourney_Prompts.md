## Style Notes

- **Consistent style across all 6**: Use `--sref` with the first generated avatar you like to lock the style for the remaining 5
- **Aspect ratio**: 1:1 for circular avatar crops
- **If too cartoonish**: Add `--s 100` and remove `cartoon` from the prompt
- **If too realistic**: Add `Pixar-inspired` or `Arcane Netflix style` to push the stylization
- **Background removal**: Use `--no background` or remove in post for transparent avatars
- **Variations**: Once you have a base you like, use `V1-V4` buttons or `--chaos 10` for subtle alternatives
