# Narrative Quality Skill

Reusable writing principles for generating high-quality narrative documents (6-pagers, strategy memos, board-ready prose).

## Trigger Phrases
- "improve narrative", "prose quality", "6-pager quality", "writing quality", "enhance document readability"

## Writing Principles (Positive Instructions)

1. Every paragraph starts with a clear topic sentence stating its controlling idea
2. Vary sentence length: alternate short declarative (8-12 words) with longer complex (20-30 words)
3. Apply the "given-new contract": begin sentences with familiar context, end with new information
4. Use active voice ("We will capture market share" not "Market share will be captured")
5. Be concrete: replace vague modifiers ("significant growth") with specific figures/timeframes from research data
6. Use em dashes for emphasis or clarifying asides
7. Use semicolons to join related independent clauses when the relationship is obvious
8. Each page reads as a self-contained argument that also advances the overall document arc

## Anti-Patterns to Avoid

- NEVER begin consecutive paragraphs with same sentence structure
- NEVER use more than once per page: "In order to", "It is important to note", "It should be noted", "Moving forward", "Going forward", "Leveraging", "In today's rapidly changing"
- NEVER use hedging: "arguably", "potentially", "it could be said that", "to some extent"
- NEVER use weak filler transitions: "Furthermore", "Moreover", "Additionally" — replace with transitions showing logical relationships: "Despite this...", "This constraint shapes...", "The same dynamic drives..."
- NEVER repeat the same adjective more than once per page
- Avoid noun clusters longer than 3 words
- Do not use rhetorical questions — state the point directly

## Cross-Page Narrative Threading

- Page 1 must end with tension or a question that Page 2 answers
- Each page begins by briefly referencing the conclusion of the previous page
- The final paragraph of each page should be the strongest — it is the last thing the reader sees
- Use the company name at least 3 times per page but not in consecutive sentences

## Quality Validation Checks

| Check | Threshold |
|-------|-----------|
| Word count per page | 350-700 words |
| Consecutive paragraph openers | Same first 3 words = warning |
| Hedging language density | >2 per page = warning |
| Passive voice ratio | >30% of sentences = warning |
| Company name usage | <2 per page = warning |
